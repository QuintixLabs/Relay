# Project Structure

- `src/` — backend code for the hub server, agent mode, routes, and system helpers.
- `public/` — static HTML, CSS, JS, and assets for the Relay app and settings page.
- `config/` — runtime instance files like `devices.json` and `settings.json`.
- `.github/` — GitHub docs, templates, and workflow files.
- `.env.example` — optional advanced environment config.
